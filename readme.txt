Version 1.0

Ein wunsch um xfire und Steam anzuzeigen. 
ist eine Vorversion. Kommen noch Erweiterungen und bessere ansicht ;)
--------------------------------------------------------------

FEATURE:

- Administrierung komplett im Adminbereich
- Vorschau der Banner welche angezeigt werden
- Festlegen ab welchem Rang man in der Ansicht angezeigt wird.
- Add-funktion damit man die Spieler welche angezeigt werden auch gleich adden kann.
--------------------------------------------------------------

HINWEIS:

Solltet ihr auf Speichen gehen ohne eine Signatur auszuwählen so wird auch keine mehr angezeigt ;)
--------------------------------------------------------------

INSTALLATION:

- alle dateien in die Ordnerstruktur hochladen
	(es werden keine vorhanden dateien überschrieben)
- installation ausführen:
	www.deine-homepage.de/index?installation
- im Adminbereich unter Module "Preprietär" aussuchen und loslegen :)
